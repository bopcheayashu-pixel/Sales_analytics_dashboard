# Sales Analytics Dashboard

## Project Overview
This project is an interactive Sales Analytics Dashboard built in Microsoft Excel.

## Features
- Total Sales KPI
- Total Profit KPI
- Sales by Region
- Monthly Sales Trend
- Top Products Analysis
- Interactive Filters (Slicers)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Data Visualization
- KPI Reporting
- Dashboard Design
